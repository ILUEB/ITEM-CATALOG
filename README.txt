Project "Item Catalog"

DESCRIPTION

This project is a web application.
It provides a list of items within a variety of categories and integrates third party user registration and authentication. 
Authenticated users should have the ability to post, edit, and delete their own items.

It is based on the Restaurant Menu Application provided by Udacity.


GENERAL FUNCTIONALITY

The web application shows a range of shops.
Selecting a shop shows you all the items the shop offers.
Selecting an item shows you the description, the price and the style category.

After logging in, a user has the ability to add, update, or delete those shops that they created.


RUNNING THE PROGRAM

1. Install Vagrant and VirtualBox.
3. Launch the Vagrant VM (by typing vagrant up in the directory fullstack/vagrant from the terminal). 
4. Run the application within the VM by typing "python application.py" into the Terminal. 
5. Visit http://localhost:8000 to see the application.
